# SpendLens AI

SpendLens AI is an AI-based mobile expense tracking application that uses receipt OCR, text classification, and spending forecasting to help users manage expenses more intelligently.

## Project Title

SpendLens AI: Mobile Expense Tracking using Receipt OCR, Text Classification, and Spending Forecasting

## Main Features

- User registration and login
- Manual expense tracking
- Receipt scanning using OCR
- Receipt text cleaning and field extraction
- Expense category prediction using machine learning
- Monthly spending prediction
- Expense history and analytics dashboard

## Tech Stack

### Mobile App
- Flutter
- Dart
- Google ML Kit

### Backend
- Spring Boot

- JWT Authentication

### Machine Learning
- Python
